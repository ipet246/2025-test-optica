Hicimos con las chicas de óptica una colaboración entre especialidades. Hicimos una serie de tests de óptica en programa para hacer pruebas y eso con las cosas que hicimos los chicos de programación.

Los tests que hicimos los chicos de programación que las chicas de óptica propusieron fueron los siguientes: 

Ishihara para hacer pruebas de daltonismo y ver si tiene dificultades para ver colores;seguimiento visual para ejercita los muculos del ojo;

Campimetria para ver los puntos ciegos o donde mas se le dificulta ver al usuario;

Tateti para jugar con alguien o un bot las X o O se ponen de color verde y rojo para que cuando se pngan filtros puedan ver esos colores nomas y trabajar la vista.
